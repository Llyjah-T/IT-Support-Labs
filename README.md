# IT Support Labs - LAN & Network Configuration

A comprehensive collection of hands-on labs for learning Local Area Network (LAN) fundamentals, network configuration, and basic firewall management.

## 📚 Lab Overview

This repository contains practical labs covering essential networking concepts including:
- **LAN Pinging** - Testing connectivity between multiple devices
- **VLAN Configuration** - Creating and managing Virtual Local Area Networks
- **Network Topology** - Building networks with switches and routers
- **Firewall Basics** - Introduction to firewall configuration and rules

## 🎯 Topics Covered

### Core Networking Concepts
- Device-to-device communication using ping utilities
- Network connectivity troubleshooting
- IP addressing and subnet management

### VLAN Labs
- Creating VLANs on network switches
- VLAN tagging and trunking
- Inter-VLAN routing

### Network Architecture
- Switch-only network topologies
- Router-based network configurations
- Multi-segment LAN design

### Firewall Introduction
- Basic firewall rules and policies
- Traffic filtering and access control
- Firewall rule management

## 🔧 Getting Started

### Prerequisites
- Basic understanding of networking concepts
- Access to network simulation software (e.g., Cisco Packet Tracer, GNS3) or physical lab equipment
- Network devices: switches, routers, computers/endpoints

### Running the Labs
1. Choose a lab from the available materials
2. Follow the step-by-step instructions
3. Configure devices as specified
4. Test connectivity and verify configurations

## 📝 Lab Exercises

Each lab includes:
- Detailed objectives
- Device configuration steps
- Verification commands (ping, ipconfig, etc.)
- Expected outcomes

## ⚙️ Requirements

### Network Equipment
- Cisco switches or equivalent
- Cisco routers or equivalent
- Multiple end devices (PCs/laptops)
- Network cables (Ethernet)

### Software Options
- Cisco Packet Tracer (free with Cisco NetAcademy)
- GNS3 (free and open-source)
- Actual lab equipment

### Knowledge Level
These labs are designed for **beginners to intermediate** learners and provide foundational knowledge in network administration and support.

## 📖 Lab Structure

Labs are organized by difficulty and topic:
- **Basic**: LAN pinging, simple device connectivity
- **Intermediate**: VLAN configuration, multi-switch networks
- **Advanced**: Router-based routing, firewall policies
